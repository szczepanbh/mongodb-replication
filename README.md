# MongoDB replicaset clasters

To create two shared-clasters containging three-member replica set each just run
```
docker-compose up
./commands.sh
```
it will create such a containers
- mongors1n1
- mongors1n2
- mongors1n3
- mongors2n1
- mongors2n2
- mongors2n3
- mongocfg1
- mongocfg2
- mongocfg3
- mongos1
- mongos2
